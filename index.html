🤖 Offline Český Hlasový Asistent

Vítejte v dokumentaci hlasového asistenta, který funguje zcela offline s využitím lokálních nástrojů a modelů, navržený speciálně pro češtinu.

Technologie & Klíčové Vlastnosti

Přehled technologií, na kterých je projekt postaven, a jeho hlavních výhod.

⚙️ Použité Technologie

  • 🧠
    LLM (llama.cpp): Pro generování odpovědí s využitím modelů jako Mistral-7B.
  • 🗣️
    TTS (Coqui TTS): Pro kvalitní převod textu na českou řeč.
  • 👂
    Wake-Word (Picovoice Porcupine): Pro bezdotykovou aktivaci na klíčové slovo.
  • 🎙️
    STT (Whisper): Pro přesný převod řeči na text v offline režimu.

✨ Klíčové Vlastnosti

  • 🔌
    Plně Offline: Nevyžaduje připojení k internetu ani cloudové API.
  • 🇨🇿
    Podpora Češtiny: Rozumí, zpracovává a odpovídá v českém jazyce.
  • 🗣️
    Detekce Wake-Word: Aktivuje se na hlasový povel.
  • 🔇
    VAD (Voice Activity Detection): Efektivně ořezává ticho z nahrávek.
  • 👐
    Open-Source: Postaveno na volně dostupných nástrojích.

🚀 Jak Začít

Podrobný průvodce, který vás provede instalací, konfigurací a spuštěním asistenta.

1

Předpoklady

Ujistěte se, že máte nainstalovány všechny potřebné nástroje.

  • Python 3.10+
  • FFmpeg
  • Git
  • Picovoice Porcupine SDK (s přístupovým klíčem)
  • Český LLaMA model ve formátu GGUF
  • Český Coqui TTS model
2

Instalace

Klonování repozitáře a instalace závislostí.

Klonování repozitáře:

git clone https://github.com/Polygonbeater/ai-assistant-voice-cs.git
cd ai-assistant-voice-cs

Instalace závislostí:

pip install -r requirements.txt
3

📥 Stáhnutí Modelů

AI modely nejsou součástí repozitáře kvůli jejich velikosti. Stáhněte si je prosím ručně.

  • LLaMA Model: Stáhněte kompatibilní GGUF model (např. `mistral-7b-instruct-v0.2.Q4_K_M.gguf`) z Hugging Face a vložte ho do složky `models/`.
  • Picovoice Porcupine: Získejte přístupový klíč z Picovoice Console. Stáhněte `.ppn` soubor pro vaše klíčové slovo a OS a vložte ho do složky `models/`.
  • Coqui TTS: Tento model se stáhne automaticky při prvním spuštění asistenta.
4

🎨 Konfigurace

Vytvořte soubor `config.json` a upravte ho podle svých potřeb. Klikněte na klíč pro zobrazení popisu.

{
  ""porcupine"": {
    "access_key": "YOUR_PICOVOICE_ACCESS_KEY",
    "model_path": "models/porcupine_params.pv",
    "keyword": "computer",
    "sensitivity": 0.7
  },
  ""whisper"": {
    "model": "medium",
    "language": "cs"
  },
  ""llama"": {
    "model": "models/mistral-7b-instruct-v0.2.Q4_K_M.gguf",
    "max_tokens": 150
  },
  ""tts"": {
    "model_name": "tts_models/cs/cv/vits",
    "gpu": false
  },
  ""audio"": {
    "device_index": -1,
    "wake_word_device_index": -1,
    "max_recording_time": 15
  },
  ""silero_vad"": {
    "sample_rate": 16000,
    "threshold": 0.3,
    "silence_duration_ms": 2000
  }
}

Klikněte na klíč v konfiguraci vlevo pro zobrazení nápovědy.

5

🏃‍♂️ Spuštění asistenta

Spusťte hlavní skript. Při prvním spuštění budete vyzváni k výběru mikrofonu.

python3 main.py

📁 Struktura Projektu

  • ai-assistant-voice-cs/
    • main.py: Hlavní smyčka a logika asistenta.
    • audio.py: Zpracování zvuku a detekce wake-wordu.
    • stt_module.py: Wrapper pro Speech-to-Text.
    • llama_module.py: Wrapper pro LLaMA model.
    • tts_module.py: Wrapper pro Text-to-Speech.
    • list_tts_models.py: Pomocný skript pro výpis TTS modelů.
    • config.json: Konfigurační soubor.
    • models/: Adresář pro AI modely.
    • requirements.txt: Seznam závislostí.
    • LICENSE: Licenční soubor.
